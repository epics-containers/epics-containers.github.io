Installation Tutorial¶

Intro¶

This project is documentation only but if you want to update the documentation and build it then follow these instructions

Check your version of python¶

You will need python 3.7 or later. You can check your version of python by typing into a terminal:

python3 --version

Use Pipenv to install developer tools¶

Install pipenv if you do not already have it https://pypi.org/project/pipenv/

pipenv install –dev

Building the documentation with sphinx¶

pipenv run docs
